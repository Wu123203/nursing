## 一、安全性优化（高优先级）

### 1.1 密码加密存储

**问题描述：**
- 当前数据库中密码以明文形式存储
- 存在严重的安全隐患

**优化方案：**
```java
// 使用 BCryptPasswordEncoder 加密密码
import org.springframework.security.crypto.bcrypt.BCryptPasswordEncoder;

@Service
public class PasswordService {
    private final BCryptPasswordEncoder encoder = new BCryptPasswordEncoder();
    
    // 加密密码
    public String encode(String rawPassword) {
        return encoder.encode(rawPassword);
    }
    
    // 验证密码
    public boolean matches(String rawPassword, String encodedPassword) {
        return encoder.matches(rawPassword, encodedPassword);
    }
}
```

**实施步骤：**
1. 添加 Spring Security 依赖
2. 创建密码加密服务
3. 修改注册/登录逻辑
4. 迁移现有密码数据

**预期收益：**
- 防止数据库泄露导致密码泄露
- 符合安全合规要求

### 1.4 SQL注入防护 ✅ 已完成

**问题描述：**
- 部分SQL语句可能存在注入风险，特别是使用 `${}` 字符串拼接的动态SQL
- 未对用户输入进行严格验证和过滤
- 缺少统一的SQL注入检测机制

**风险等级：** 高危

**已实现方案：**

#### 1.4.1 MyBatis 参数绑定规范

**禁止使用字符串拼接 `${}`**，必须使用预编译参数 `#{}`。

#### 1.4.2 动态SQL安全写法

使用 MyBatis 提供的安全方式处理动态表名、排序字段等场景。

#### 1.4.3 输入验证与过滤

为 `LoginDto` 添加 JSR-380 验证注解（@NotBlank、@Size、@Pattern）。

#### 1.4.4 SQL注入检测拦截器

创建 `SqlInjectInterceptor`，检测常见 SQL 注入关键词。

#### 1.4.5 数据库层面防护

创建最小权限数据库用户，限制危险操作权限。

**实现文件：**
- `common/interceptor/SqlInjectInterceptor.java` - SQL注入检测拦截器
- `common/dto/LoginDto.java` - 添加验证注解
- `common/config/MyBatisConfig.java` - MyBatis配置

- 存在跨域攻击风险

**优化方案：**
```java
@Configuration
public class CorsConfig implements WebMvcConfigurer {
    
    @Override
    public void addCorsMappings(CorsRegistry registry) {
        registry.addMapping("/**")
            .allowedOriginPatterns("http://localhost:*", "https://yourdomain.com")
            .allowedMethods("GET", "POST", "PUT", "DELETE", "OPTIONS")
            .allowedHeaders("*")
            .allowCredentials(true)
            .maxAge(3600);
    }
}
```

```java
    private String msg;     // 消息
    private T data;         // 数据
    private Long timestamp; // 时间戳
```

| 点赞功能 | 使用 Redis Set 防重复点赞，INCR/DECR 计数 |
| JWT Token 黑名单 | 登出时将 Token 加入 Redis 黑名单，支持主动失效 |
| 定时任务 | 使用 @Scheduled 同步 Redis 数据到数据库 |

**实现文件：**
- `common/interceptor/RateLimitInterceptor.java` - 接口限流拦截器
- `common/annotation/RateLimit.java` - 限流注解
- `common/lock/DistributedLock.java` - 分布式锁工具类

- `service/impl/LikeServiceImpl.java` - 点赞功能

**预期收益：**
- ✅ 防止接口被恶意刷取，保护系统稳定性
- ✅ 杜绝重复提交问题，保证数据一致性
